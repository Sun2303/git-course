What is the Git command to set up email and name?
***For All projects:
- for email: git config --global user.email "sun.libra95@gmail.com"
- for name: git config --global user.name "Sun"

***For a specified project (current project):
- for email: git config user.email "sun.libra95@gmail.com"
- for name: git config user.name "Sun"
